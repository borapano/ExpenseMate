# Application Concept – ExpenseMate

## Application  
Web-based shared expense management system for groups.

## Core Features  
- User registration and authentication (login/signup)  
- Create or join groups using a unique group code  
- Add expenses with amount, description, payer, and participants  
- Flexible expense splitting (equal or custom participants)  
- Edit and delete only user-created expenses  
- Automatic balance calculation between group members  
- Clear summary of who owes whom  

## Users  
- Group Members: track shared expenses and settle debts  
- Group Creators: manage and organize group activities  
- Admins (optional): oversee system usage and data  

## Constraints  
- Ensure data consistency during concurrent updates  
- Secure authentication and user data protection  
- Scalable for multiple groups and active users  
- Responsive design (desktop and mobile)  
- Accurate real-time balance calculations
