# Monolithic Architecture – ExpenseMate

- **Frontend App**: Web interface for users to create/join groups, add expenses, and view balances.  
- **Authentication Module**: Manages user registration, login, and sessions.  
- **Group Management Module**: Handles creation of groups, joining with group codes, and managing group members.  
- **Expense Management Module**: Allows adding, editing, and deleting expenses within a group.  
- **Balance Calculation Module**: Automatically computes who owes whom and provides summaries of debts.  
- **Reporting Module**: Generates summaries and overviews of group expenses and balances.  
- **Database**: Central storage for all user, group, and expense data.
