# Project Documentation - ExpenseMate

## Purpose
To provide a sophisticated, high-performance platform for groups to manage shared expenses. ExpenseMate simplifies financial interactions through a premium user experience, automating debt calculations and eliminating social friction through elegant, real-time reporting.

## Users
* **Roommates:** Seamlessly track shared rent, utilities, and household essentials.
* **Travelers:** Manage multi-currency group costs for flights, stays, and dining.
* **Event Organizers:** Coordinate budgets for high-stakes events like weddings or galas.
* **Social Circles:** Effortlessly settle casual tabs with a few taps.

## Core Screens
* **Global Dashboard:** A premium high-level overview featuring the "Net Balance" summary, utilizing the Deep Navy and Gold highlight palette for clarity.
* **Group Workspace:** A refined transaction feed with integrated "Settle Up" logic and group-specific debt summaries.
* **Expense Entry Form:** An intuitive, multi-step interface for logging amounts, descriptions, and custom split-logic (Even, Percentage, or Exact).
* **Analytics Hub:** Premium data visualizations (Pie/Bar charts) reflecting spending trends across categories and timeframes.

## Actions
* **Group Management:** Instant group creation with unique `Invite Codes` and simplified member onboarding.
* **Expense Logging:** Smart expense distribution that auto-calculates `Expense_Participant` shares in real-time.
* **Debt Simplification:** Advanced algorithms to minimize the number of required transactions (e.g., Alice settles with Charlie directly).
* **Smart Filtering:** Deep-dive into financial history by category, date, or specific member contributions.
